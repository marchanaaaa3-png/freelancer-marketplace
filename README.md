# Freelancer Marketplace 💼

A modern full stack Freelancer Marketplace platform where clients can post projects and freelancers can apply for jobs.

# Features
- User Login & Signup
- Freelancer Profiles
- Project Posting System
- Job Applications
- Responsive UI
- Modern Black & White Design
- Full Stack Functionality

# Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

# Project Overview
This platform connects freelancers and clients in a simple and modern interface. Clients can post projects and freelancers can browse opportunities and apply directly.
